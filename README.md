## Overview
Static webpage built for interview assessment.

## Setup
### Prerequisites
- Node.js (v14 or higher)
- npm (comes with Node.js)
### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/nightZQ/Assessment.git
   cd Assessment
   ```
2. Install dependencies
	```bash
	npm install
	```
### Getting Started
1. Run the following commands:
	```bash
	npm run build #Build & minify CSS
	npm start #Start development server
	```
2. Open http://localhost:8080 (or other port based on availability)
